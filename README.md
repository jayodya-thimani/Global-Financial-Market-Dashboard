# 📌 Project Overview

The **Global Financial Market Dashboard** is an interactive Business Intelligence solution developed using **Microsoft Power BI** to analyze historical financial market data from **2010–2021**.

This dashboard enables users to explore market performance across multiple financial assets including:

- 📈 NIFTY 50
- 📈 S&P 500
- 🥇 Gold
- 🛢 Brent Oil
- 💱 USD/INR Exchange Rate

The report provides interactive visualizations, KPI tracking, forecasting, AI-powered insights, and drill-through analysis to support financial decision-making.

---

# 🎯 Project Objectives

- Analyze historical financial market trends.
- Compare the performance of major global financial assets.
- Monitor key financial indicators using KPI cards.
- Explore market behaviour using interactive dashboards.
- Generate AI-powered insights.
- Forecast future market trends using Power BI Forecast Analytics.
